## Growple 스터디 개설 화면 UI/UX 레이아웃 구성도

스터디 개설 화면은 **스터디장이 되고자 하는 사용자가 스터디의 모든 정보를 입력하고 개설하는 폼 화면**입니다. 입력 과정이 복잡해 보이지 않도록 **단계별 진행 방식**을 제안합니다.

---

### 1. 전체적인 디자인 컨셉 및 톤앤매너

- **키워드**: 명확함, 가이드, 단계별 진행, 완성도
- **메인 컬러**: 메인 화면과 동일한 초록색 계열 유지
- **분위기**: 복잡한 정보 입력을 쉽고 직관적으로 느끼게 하는 친절한 폼 UI

---

### 2. 상단 헤더 (GNB)

메인 화면과 동일하게 구성합니다.

- **로고**: Growple
- **메인 메뉴**: 스터디 찾기, 스터디 만들기
- **우측**: 유저 스테이터스 바 (Lv, 포인트, 알림, 프로필)

---

### 3. 진행 단계 표시 (스테퍼)

사용자가 현재 어느 단계에 있는지 시각적으로 보여줍니다.

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│    ①────────②────────③────────④────────⑤                                │
│  기본 정보    일정 설정    참가 설정    상세 설명    미리보기               │
│    ✓           ●           ○           ○           ○                      │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

- 완료된 단계: ✓ (체크 표시, 초록색)
- 현재 단계: ● (활성화, 메인 컬러)
- 미완료 단계: ○ (비활성화, 회색)
