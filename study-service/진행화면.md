# Growple 스터디 진행 화면 상세 설계

---

## 1. 화면 개요

### 접근 조건

- **참가자만 접근 가능** (참가비 결제 완료 후)
- 스터디 시작일 이후부터 활성화
- 스터디 종료 후에도 일정 기간(30일) 열람 가능 (읽기 전용)

## 3. 스터디 헤더 영역

항상 상단에 고정되어 현재 스터디 상태를 한눈에 파악할 수 있게 합니다.

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│  📚 매일 아침 7시 기상 미라클 모닝 & 알고리즘 1문제 풀기                        │
│                                                                             │
│  [개발]  [초급]                                                              │
│                                                                             │
│  ┌──────────────────┐  ┌──────────────────┐  ┌──────────────────┐          │
│  │ 📅 진행 현황      │  │ ⏰ 다음 스터디    │  │ 🎯 내 출석률      │          │
│  │ 2주차 / 4주차     │  │ 내일 07:00       │  │ 85% (12/14회)    │          │
│  │ ████████░░ 50%   │  │ D-1              │  │ 예상환급 30,000P │          │
│  └──────────────────┘  └──────────────────┘  └──────────────────┘          │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

### 표시 정보

- 스터디 제목, 카테고리, 난이도
- 진행 현황 (n주차/총 n주차, 프로그레스 바)
- 다음 스터디 일정 (요일, 시간, D-day)
- 내 출석률 및 예상 환급 금액
